Q2. Explain the architecture of JDK in details
Soln:
JDK (Java Development Kit) contains everything needed to develop Java programs.

# Components:

1. JDK
It Provides/ Includes
   * Development tools
   * Compiler (javac)
   * Debugger

2. JRE (Java Runtime Environment)

   * It Provides environment to run programs

3. JVM (Java Virtual Machine)

   * It Executes bytecode

# Flow:

JDK → JRE → JVM
JDK = JRE + Tools
JRE = JVM + Libraries


Q3. What is JVM? Explain its role in Java
Soln:
JVM (Java Virtual Machine) is responsible for executing Java programs.

# Role:

* Converts bytecode --> machine code
* Makes Java platform independent
* Manages memory (heap, stack)
* Handles garbage collection

Q4.Explain the process of "Write Once, Run Anywhere"
Soln:
Java program is written once and runs anywhere.

# Process:

1. Write code (.java)
2. Compile → bytecode (.class)
3. JVM executes bytecode on any OS
Same file runs everywhere --> platform independence

Q5. Explain the concept of bytecode in Java
Soln: 
Bytecode is intermediate code generated after compilation.

* Generated by javac
* Stored in .class file
* Not machine-specific
* Executed by JVM

 Type     Default Value

Q10.A Shopkeeper records daily sales for a week, find the total sales and highest sale day
Soln:


class Sales {
    public static void main(String[] args) {

        int[] sales = {100, 200, 150, 300, 250, 400, 350};

        int total = 0;
        int max = sales[0];
        int day = 1;

        for(int i = 0; i < sales.length; i++) {
            total += sales[i];

            if(sales[i] > max) {
                max = sales[i];
                day = i + 1;
            }
        }

        System.out.println("Total Sales: " + total);
        System.out.println("Highest Sale: " + max);
        System.out.println("Day: " + day);
    }
}
