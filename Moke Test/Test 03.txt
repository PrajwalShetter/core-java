soln:
Method is a block of code. used to perform a particular task.


Q4. what is method Overloading?
soln:
In a class two or more methods having a same name with different parameters is called Method Overloading.

Q5. what is a constructor?
soln:
A constructor is a special method. it has same name as class name and used for initializing the instance variables.

Q7. what is this keyword in java?
soln:
this keyword refers to the same class. it is mainly used to initialize or invoke the methods and variables in the same class.
