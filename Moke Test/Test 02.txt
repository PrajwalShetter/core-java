1) Explain Constructor chaining using this() with example
soln:
* constructor is a special method it used to initialize the instance variables.
* The first line of the constructor always super() -- it invokes the parent class of the constructor

Ex.
class Person{
 String name;
 int age;

Person(){
super();
this.name = name;
this.age = age;

}

}


2) Explain default values assigned to instance variables in java with example
soln:
* Default values of instance variables always depends upon the datatype of the variable
* datatype --- default values
   String  ---  null
   integer --- 0
   double  --- 0.0
   Boolean --- false
etc

Ex.
class Person{

String name;
int age;
Boolean isMale;


Person(){
this.name = name;
this.age= age;
this.isMale = isMale;
System.out.println("name :"+this.name);
System.out.println("age :"+this.age);
System.out.println("isMale :"+this.isMale);
}

public static void main(String[] args){

Person person = new Person();

}
}
